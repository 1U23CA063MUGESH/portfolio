# My Portfolio

A responsive and user-friendly portfolio website built with Bootstrap 5, featuring separate pages for different sections.

## Features

- **Multi-page Design**: Separate HTML files for Home, About, Skills, Projects, and Contact
- **Responsive Design**: Fully responsive layout that works on all devices using Bootstrap 5
- **Modern UI**: Clean, professional design with gradient backgrounds and card-based layouts
- **Interactive Elements**: Form validation, hover effects, and smooth transitions
- **Frontend Focus**: Showcases frontend development skills and projects

## Pages

- **index.html**: Welcome page with hero section and navigation
- **about.html**: About me section for a UG fresher frontend developer
- **skills.html**: Skills showcase with progress bars (HTML, CSS, JavaScript, Bootstrap, MySQL, React JS)
- **projects.html**: Frontend-related projects with tech stacks
- **contact.html**: Contact form with client-side validation

## Technologies Used

- HTML5
- CSS3
- Bootstrap 5
- JavaScript (ES6+)
- Font Awesome (for icons)

## How to View

1. Open any of the HTML files directly in your web browser
2. For the best experience, use a local web server (e.g., Live Server extension in VS Code)
3. Start with `index.html` for the home page

## Customization

- Replace placeholder images with your actual photos/projects
- Update the content in each section to reflect your information
- Modify colors and styles in the `<style>` sections
- Add real links to your project repositories and social media
